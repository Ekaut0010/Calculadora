# 🧮 Calculadora Web

Uma calculadora simples feita com HTML, CSS e JavaScript, com suporte a tema claro e escuro.

## 🚀 Funcionalidades

- Operações básicas (+, -, x, /)
- Tema claro e escuro 🌙☀️
- Interface responsiva

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
